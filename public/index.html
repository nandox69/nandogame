Bienvenido a Nando Game

Estimado usuario,

Le damos la bienvenida a la plataforma web de Nando Game. Este sitio se encuentra actualmente en fase activa de desarrollo, por lo que su acceso está destinado exclusivamente a pruebas y evaluación de rendimiento.

Al interactuar con nuestra interfaz web, usted acepta las siguientes condiciones de seguridad y uso legítimo:

Política de Uso y Sanciones
El sistema debe utilizarse únicamente para los fines de prueba asignados. Cualquier intento de alteración del código, inyección de scripts, explotación de fallos o mal uso de las herramientas web resultará en la suspensión inmediata y el bloqueo permanente de su usuario y credenciales de acceso.

Registro de Actividad y Auditoría Web
Para garantizar la integridad del servidor, toda la navegación y actividad dentro de esta web está siendo estrictamente monitoreada. Nuestra base de datos registra de forma automatizada:

Dirección IP de origen y geolocalización de red.
Información del dispositivo (Navegador web, sistema operativo y resolución).
Tiempo de uso (Duración de la sesión, marcas de tiempo de entrada y salida).
Modificaciones del sistema (Cualquier cambio de datos, peticiones HTTP o alteración en la base de datos).

Declaración de Consentimiento: Al hacer clic en "Entendido y Aceptar", usted declara estar en pleno conocimiento de estos términos y autoriza el registro de los datos antes mencionados para fines de auditoría técnica.

Agradecemos su colaboración para mantener un entorno de desarrollo seguro.
